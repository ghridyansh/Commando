Vulnerability Tester

Vulnerability Testing, is a software testing type performed to evaluate the security risks in the software system in order to reduce the probability of a threat. A vulnerability is any mistakes or weakness in the system security procedures, design, implementation or any internal control that may result in the violation of the system's security policy. In other words, the possibility for intruders (hackers) to get unauthorized access.
